Installation

  1. create a working directory and cd into it

    2. mkdir vanilla cd vanilla

  2. degit from gh in dir
    3. pip install git+https://github.com/vanilla-templates/vt.git
  3. create .gitignore in working directory and add into it.
  4. create public directory (NOTE: We insist you call it public) for static files.

    5. mkdir public

  5. create an index.html and start creating vanilla templates
  6. Build or serve your work on a configured port.
    python -m vt.serve -- for serving
    python -m vt.building -- for generating static site in the build directory
